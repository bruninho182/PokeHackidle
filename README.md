# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/SEU_USUARIO/PokeHackIdle.git
```

Open Chrome Extensions

```
chrome://extensions
```

Enable **Developer Mode**

Click

```
Load unpacked
```

Select the project folder.

Done! 🎉

---

# 🎮 How to use

1. Open **Poke Idle World**

2. Click the extension icon.

3. Configure the desired options.

4. Start hunting.

The extension automatically monitors:

- Pokémon captures
- Rare Pokémon
- High IV Pokémon
- Item drops
- Statistics
- Capture history
